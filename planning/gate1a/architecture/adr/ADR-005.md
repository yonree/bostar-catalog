# ADR-005：Route and Redirect Manifest

- status: `PROPOSED`

## context

126 条生产 URL 需要保留、301、恢复或待确认决策。

## decision

用显式 manifest 管理目标 URL 和 redirect 决策。

## alternatives

散落在 `next.config` 或页面逻辑。

## consequences

URL 变更可审计、可测试。

## risks

manifest 与实现不同步。

## rollback

以只读 manifest 回退，不立即激活生产规则。
