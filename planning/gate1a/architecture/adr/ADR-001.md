# ADR-001：保留 Next.js App Router

- status: `PROPOSED`

## context

现有仓库已使用 App Router，Gate 0C 干净环境可正常 build/start。

## decision

继续使用 Next.js App Router，避免在重构期间迁移框架。

## alternatives

切换 Pages Router；切换到其他 React 框架。

## consequences

保留现有路由能力与服务器组件模型。

## risks

历史页面逻辑与新模板混用期间需要额外边界控制。

## rollback

如 Gate 2 发现 App Router 无法支撑需求，再以新 ADR 评估。
