# ADR-003：Metadata Factory

- status: `PROPOSED`

## context

当前存在 124 条首页 canonical 异常，metadata 逻辑分散。

## decision

引入统一 Metadata Factory 生成 title、description、canonical、robots、OG、alternates。

## alternatives

各页面手写 metadata。

## consequences

规则集中，可测试。

## risks

迁移初期页面 metadata 接口不统一。

## rollback

允许页面保留局部覆盖，但统一入口不移除。
