# Component Spec

## 基础组件

- Button：primary / secondary / ghost / danger
- Input / Textarea / Select：统一 48px 高度、明确错误态
- Table：参数表优先双列或多列稳定布局
- Badge：只用于状态或类型，不作装饰
- Breadcrumb：最多 4 层

## 业务块

- 产品参数模块：标题、分组、单位、备注
- 下载模块：标题、文件类型、版本、是否需要留资
- 文章模块：作者/审核/发布时间/更新时间/来源
- CTA 模块：单主按钮 + 可选次按钮

## 禁止

- 嵌套卡片
- 仅为视觉装饰创建无意义容器
- 把技术参数塞进轮播或手风琴首屏之外
