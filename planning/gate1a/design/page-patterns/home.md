# 首页样板

- 页面目标：快速建立品牌可信度，并把用户分流到产品、方案、知识和询盘。
- 目标用户：采购、工程师、企业决策者。
- 信息优先级：品牌价值 > 产品入口 > 方案入口 > 技术可信度 > 下载/询盘。
- 区块顺序：Hero -> 产品分类 -> 解决方案 -> 技术亮点 -> 下载/资料 -> 联系 CTA。
- 桌面布局：首屏双列，左文本右产品实图；中段 3 列分类网格。
- 移动布局：单列堆叠，保留首屏 CTA 和分类快捷入口。
- 数据字段：品牌、主标题、描述、产品分类、精选方案、下载入口、联系信息。
- 组件清单：Header、Hero、CategoryGrid、SolutionHighlights、DownloadCTA、Footer。
- CTA：`获取技术方案`、`浏览产品`。
- SEO 要素：品牌级 title/description、自引用 canonical。
- Schema：Organization、WebSite、BreadcrumbList。
- 无障碍要求：首屏 CTA 可键盘触达，装饰图空 alt。
- 加载和错误状态：精选区块缺数据时回退为静态入口，不隐藏联系 CTA。
- 空数据状态：分类为空时显示固定“浏览产品中心”入口。
- 验收检查点：3 次主要交互内可到产品详情或联系表单。
