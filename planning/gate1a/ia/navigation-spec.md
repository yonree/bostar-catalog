# Navigation Spec

## Header

- Desktop 一级导航：产品与设备、涂装解决方案、核心技术、应用行业、知识与服务、资料下载、关于博士达、联系与询盘
- Header CTA：`获取技术方案` 或 `提交询盘`
- 一级导航原则不超过 8 项，不显示后台、搜索结果、404 等内部页面。

## Mega Menu / 下拉

- 产品与设备：展示产品分类，不直接展开全部型号。
- 涂装解决方案：按场景/工艺分组。
- 知识与服务：知识分类、FAQ、服务、视频、新闻。

## Mobile Nav

- 一级项折叠为手风琴。
- 同一时刻只展开一个一级大类。
- 联系 CTA 固定在导航底部，不埋进二级列表。

## Footer

- 仅保留：主导航摘要、重点产品分类、下载入口、公司联系信息。
- 不承载无归属页面。

## Breadcrumb

- 首页 > 一级栏目 > 二级栏目/分类 > 详情页
- 3 层为常态，4 层只用于文章或型号详情。

## Search

- 搜索入口为全站工具，不进入一级导航层级统计。

## 禁止项

- 不允许把历史孤立页仅藏在 Footer。
- 不允许把产品型号直接放满一级导航。
- 不允许语言切换把用户直接送回首页。
