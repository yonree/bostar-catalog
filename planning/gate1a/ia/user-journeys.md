# User Journeys

## Journey 1：工业采购找设备

1. 首页 / 产品与设备
2. 产品分类页
3. 产品详情页
4. 下载资料 / 提交询盘

目标：3 次主要交互内到达询盘入口。

## Journey 2：工程师按工艺找方案

1. 首页 / 涂装解决方案
2. 方案详情页
3. 关联设备详情
4. 联系工程师

## Journey 3：维护人员查故障与文档

1. 知识与服务
2. 知识文章或 FAQ
3. 下载资料
4. 服务支持或联系页

## Journey 4：海外客户切换英文

1. 当前中文页面
2. 语言切换
3. 进入对应 `/en/` 页面
4. 保持同一实体和信息层级

## Journey 5：从外链进入历史 URL

1. 历史 URL 访问
2. 命中 `KEEP_200` / `REDIRECT_301` / `RESTORE_200`
3. 保持目标内容与原主题一致

禁止：任何历史产品页统一回首页。
